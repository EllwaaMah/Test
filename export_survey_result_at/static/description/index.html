Survey : Export Result in Excel

Easy to export answers in Excel format!

Survey : Export Answers in Excel module allows you to export survey answers in Excel file. You can filter export by only completed surveys also. You can also export answers as a score if score is set for given answer. It will be easy to search, filter and process exported answers in Excel file as you want.

Odoo survey modules provide all functionality to take survey. We can create survey with multiple types of answer types. It can be multiple choice single answer, multi anser. Single line text, date or number.

But in odoo you can only view answers which are submitted. Export feature of odoo give you data in row format by answers. In our module, we export one answer by one row with all questions columned.

  • Configuration
Assign user survey administrator access rights to see export button on survey.
  • Usage
Go to survey form view that you want to export answers. On screen, you will find button "Export Answers"
When you click on button, wizard will appear with options to filter survey answers.
Only Completed Surveys: It will only consider survey answers which are completed status. It will filter New or In Progress survey answers.
Convert answers into points: Instead of answer, it will export score in Excel file. Like if answer is A and its score is 1. Then in file, it will write 1(score) instead of A(Answer).
Download exported answer's Excel file. Name of the Excel file with be survey title.
